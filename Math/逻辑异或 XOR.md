
> [[命题 Statement|命题]] $A \oplus B$ 为真，当命题A 是真的 或 命题B 是真的，但并非两个都是真的。
> 也称为排斥的或（Exclusive Or）A 或者 B （但不都满足）

### 符号表示
$$ A \oplus B \quad (A\ XOR\ B) $$

### 真值表
|   A   |   B   | $A \wedge B$ |
|:-----:|:-----:|:------------:|
| True  | True  |     False     |
| True  | False |     True     |
| False | True  |     True     |
| False | False  |     False     |
