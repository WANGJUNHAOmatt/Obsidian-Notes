> 一个空命题，就是 没有传递任何信息的命题。

#### 用程序来举例子
 ``` python
 if False:
     do_something() # 并没有执行
```

### 空命题并不是没有用的命题

我们可以基于一个空命题来推导出一个有用的结果。
