> [[命题 Statement|命题]] $A \vee B$ 为真，只需满足 命题A 与 命题B 中至少有一个为 真 即可。

### 符号表达
$$ A \vee B \quad (A\ or\ B) $$
### 真值表
|   A   |   B   | $A \wedge B$ |
|:-----:|:-----:|:------------:|
| True  | True  |     True     |
| True  | False |     True     |
| False | True  |     True     |
| False | False  |     False     |
