## 合取 逻辑与 并且
>  [[命题 Statement|命题]] $A \wedge B$ 为 真，**当且仅当** 命题A 为真 **并且** 命题B 也为真。

### 符号表示
$$ A \wedge B \quad (A\ and\  B)$$

### 真值表
|   A   |   B   | $A \wedge B$ |
|:-----:|:-----:|:------------:|
| True  | True  |     True     |
| True  | False |    False     |
| False | True  |    False     |
| False | False  |    False     |
