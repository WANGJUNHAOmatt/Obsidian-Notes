
## 课程考核
课程考核有三部分
- E - R图
- 规范化：0NF - 3NF
	- 目的：
		- 概念单一化
	- 1NF
		- 单一的二维表
		- 函数依赖
		- 找主码
	- 2NF
		- 完全函数依赖
			- 去掉部分函数依赖
	- 3NF
		- 去掉传递依赖
- SQL (Structure Query Language)
	- 数据的操纵 CRUD
		- Insert
		- Update
		- Delete
		- Selete
	- 数据结构的 CUD
		- Create
		- drop
		- alter
	- 数据控制 data control
		- GRANT
		- REVOKE
