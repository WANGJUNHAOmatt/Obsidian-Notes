当且仅当，也称双条件语句、双向蕴涵。

> 令 p 和 q 是 [[命题 Statement|命题]]。双条件语句 $p \leftrightarrow q$ 等价于 $p \implies q \wedge q \implies p$，即当 $p$ 与 $q$ 有同样的真值时，双条件语句为真。

### 真值表

|   $p$   |   $q$  | $p \leftrightarrow q$ |
|:-----:|:-----:|:------------:|
| True  | True  |     True     |
| True  | False |     False     |
| False | True  |     False     |
| False | False  |     True     |
