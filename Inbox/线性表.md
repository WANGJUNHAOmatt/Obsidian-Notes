
抽象数据类型 Abstract Data Type

`ADT List {`
数据对象：
$D = {a_i | A_i \in ElemSet, i = 1, 2, \cdots, n, n \geq 0}$

数据关系：
$R_1 = {<a_{i-1}, a_i> |a_{i-1}, a_i \in D, i = 2, 3, \cdots, n}$

操作：
- 结构初始化操作
	- `InitList(&L)`
- 结构销毁操作
	- `DestoryList(&L)`
- 引用型操作
	- `ListEmpyt(L)`
	- `ListLength(L)`
	- `PriorElem(L, cur_e, &pre_e)`
	- `NextElem(L, cur_e, &next_e)`
	- `GetElem(L, i, &e)`
	- `LocateElem(L, e, compare())`
	- `ListTraverse(L, visit())`
- 加工型操作
	- `ClearList(&L)`
	- `PutElem(&L, i, &e)`
	- `ListInsert(&L, i, e)`
	- `ListDelete(&L, i, &e)`
`} ADT List`
